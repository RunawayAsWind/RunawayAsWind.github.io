# Use Mali Offline Compiler to Test Shader Performance

Mali Offline Compiler[^OfficialDoc] is an official development tool provided by Arm for its Mali series GPUs. Its core function is to pre-compile OpenGL ES or Vulkan shader code into Mali GPU binaries on your computer and generate detailed performance analysis reports.

[^OfficialDoc]: Official documentation：https://dev.epicgames.com/documentation/en-us/unreal-engine/custom-material-expressions-in-unreal-engine
